HTML

HTML atau Hypertext Markup Language adalah bahasa markup yang digunakan untuk membuat struktur halaman website agar dapat ditampilkan pada web browser.

css

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML.

javascript

JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website,agar lebih dinamis dan interaktif
